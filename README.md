# nba-stats-engine
An automated data extraction framework designed to retrieve comprehensive NBA player and game statistics. Built to bypass anti-bot protections and emulate human-like behavior, ensuring reliable data collection from official sources.
### Core Capabilities
- **Dynamic Data Extraction:** Parses both box scores and advanced metrics directly from complex SPA interfaces.
- **Anti-Detection:** Implements human-like navigation patterns and stealth headers to maintain session integrity.
- **Reliable Data Flow:** Captures raw API responses via network interception for high-accuracy, structured data output.
### Technical Stack
Playwright (Node.js), TypeScript, JSON/CSV normalization.
